Javascript Objects:

Objects have properties e.g. weight, color, length and height. Properties and values are associated with objects. Methods are actions objects can perform.

a basic object:

  • var person = {
    firstName:"John",
    lastName:"Doe",
    age:50,
    eyeColor:"blue"

Accessing object properties:

  • person.lastName;
    person["lastName"];

Accessing object methods:

  • name = person.fullName();

Adding a Property to an Object:

  • myFather.nationality = "English";

Adding a Method to an Object:

  • myFather.name = function () {
    return this.firstName + " " + this.lastName;
    };

Javascript Prototypes

Creating a prototype using an object constructor:

  • function person(first, last, age, eyecolor) {
    this.firstName = first;
    this.lastName = last;
    this.age = age;
    this.eyeColor = eyecolor;
    }
  • var myFather = new person("John", "Doe", 50, "blue");
  • var myMother = new person("Sally", "Rally", 48, "green");

      • Adding Properties to a Prototype:

      Properties must be added to the constructor function

      • function person(first, last, age, eyecolor) {
        this.firstName = first;
        this.lastName = last;
        this.age = age;
        this.eyeColor = eyecolor;
        this.nationality = "English"
        }

      Adding Methods to a Prototype:

      The construstor function can also define methods:

      • function person(first, last, age, eyecolor) {
        this.firstName = first;
        this.lastName = last;
        this.age = age;
        this.eyeColor = eyecolor;
        this.name = function() {return this.firstName + " " + this.lastName;};
        }

      Loops in Javascript

      The for loop:
      • for (statement 1; statement 2; statement 3) {
        code block to be executed
        }
      • for (i = 0; i < 5; i++) {
        text += "The number is " + i + "
        ";
        }

        The while loop

      • while (condition) {
        code block to be executed
        }
      • while (i < 10) {
        text += "The number is " + i;
        i++;
        }
      • do {
        code block to be executed
        }
        while (condition);
      • do {
        text += "The number is " + i;
        i++;
        }
        while (i < 10);
