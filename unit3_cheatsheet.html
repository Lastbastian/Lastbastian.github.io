Javascript Objects:

Objects have properties e.g. weight, color, length and height. Properties and values are associated with objects. Methods are actions objects can perform.

a basic object:

  • var person = {
    firstName:"John",
    lastName:"Doe",
    age:50,
    eyeColor:"blue"

Accessing object properties:

  • person.lastName;
    person["lastName"];

Accessing object methods:

  • name = person.fullName();

Creating a prototype using an object constructor:
  • function person(first, last, age, eyecolor) {
    this.firstName = first;
    this.lastName = last;
    this.age = age;
    this.eyeColor = eyecolor;
    }
  • var myFather = new person("John", "Doe", 50, "blue");
  • var myMother = new person("Sally", "Rally", 48, "green");
